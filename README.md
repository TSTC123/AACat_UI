- AAcat_UI 项目介绍

  AAcat_UI 是一个基于 **Qt Widgets** 框架开发的桌面应用程序，版本 0.1。它提供了一个美观的深色主题界面，集成了多种常用控件和交互功能，适用于日常任务管理、信息记录和轻量级聊天模拟。所有用户数据（头像、用户名、轮回计数）都会自动保存，下次启动时恢复。

  ## 主要功能

  - **深色主题界面**：统一的深色配色方案，控件样式精心设计，视觉舒适。
  - **左侧固定面板**：
    - **头像**：可点击更换图片，自动裁剪为圆形并保存。
    - **用户名**：点击或使用编辑按钮修改，实时保存。
    - **轮回计数器**：记录“已过轮回”和“总轮回”，支持加减按钮调整，也可双击数值直接输入；数值变化自动保存。
    - **导航按钮**：首页、个人资料、消息、设置、帮助等快捷入口。
  - **右侧可滚动区域**：
    - **功能按钮列**：放置“切换 model”、“切换 theme”等功能按钮（界面预留，可扩展）。
    - **聊天框**：模拟聊天界面，支持发送消息、清空聊天，并带有小猫自动回复（延迟模拟）。
    - **小日历**：显示当前日期，支持选择日期。
    - **进度条**：展示多个任务的完成进度。
    - **滑块控制**：音量、亮度调节滑块，实时显示数值。
    - **复选框与单选按钮**：启用通知、自动保存、深色模式等选项。
    - **下拉选择框**：多个选项分类。
    - **任务列表**：QListWidget 展示待办事项。
    - **数据表格**：QTableWidget 展示人员信息。
  - **设置持久化**：使用 QSettings 保存用户名、头像、轮回计数，窗口关闭时自动存储。

  ## 技术栈

  - **Qt 版本**：6.8.1（兼容 Qt6 系列）
  - **编程语言**：C++17
  - **构建系统**：CMake 3.20+
  - **主要模块**：Qt Widgets

  ## 构建指南

  ### 环境要求

  - **Qt 6.8.1** 或其他版本（需包含 Widgets 模块）
  - **CMake** 3.20 或其他版本
  - 支持 C++17 的编译器（如 MSVC 2019+、GCC 9+、Clang 10+）

  ### 构建步骤

  #### 使用 Qt Creator（推荐）

  1. 打开 Qt Creator，选择“打开项目”，定位到项目根目录下的 `CMakeLists.txt`。
  2. 选择合适的构建套件（Kit），确保指向 Qt 6.8.1或其他版本 的编译器。
  3. 点击“配置项目”，等待 CMake 完成配置。
  4. 点击左下角的“构建”按钮（或按 `Ctrl+B`）生成可执行文件。

  ## 注意事项

  - 头像图片仅支持常见格式（PNG、JPG、BMP、GIF），建议使用正方形图片以获得最佳显示效果。
  - 聊天功能仅为模拟演示，无网络通信，自动回复逻辑简单。
  - 所有按钮功能已连接信号槽，但部分（如“切换 model”）尚未实现具体逻辑，可根据需要扩展。
